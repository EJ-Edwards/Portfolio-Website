Port Intel

Port Intel is a simple port scanning website that detects open, closed, and filtered ports on a target system. The backend is built with Node.js and Express, while Firebase handles user authentication for secure access. The frontend uses HTML, CSS, and JavaScript to provide a user-friendly experience.
